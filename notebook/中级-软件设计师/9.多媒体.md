## 1. 音频相关概念

![音频相关概念](../../img/软考/音频相关概念.jpg)
- 总结

## 4. 多媒体相关计算

![多媒体相关计算](../../img/软考/多媒体相关计算.jpg)
- 总结
	- 图像容量计算
		- 容量 = 像素数 x 每个像素位数
	- 音频容量计算
		- 容量 = 采样频率 x 采样位数 x 声道数 / 8
	- 视频容量计算
		- 容量 = 每帧容量 x 每秒帧数 x 时间 + 音频容量 x 时间
